# 🖼️ AI Image Description using Amazon Rekognition & Amazon Polly

## 📌 Project Overview

AI Image Description is a serverless AWS application that allows users to upload an image through a web interface. The uploaded image is stored in Amazon S3, analyzed using Amazon Rekognition, converted into a spoken description using Amazon Polly, and all related information is stored in Amazon DynamoDB.

The application displays the uploaded image, AI-generated description, detected labels with confidence scores, and an audio player to listen to the generated description.

---

# 🚀 Features

* Upload image through a web interface
* Store uploaded image in Amazon S3
* Detect objects using Amazon Rekognition
* Generate an image description
* Convert description into speech using Amazon Polly
* Store generated MP3 in Amazon S3
* Store image metadata in Amazon DynamoDB
* Display uploaded images with description and audio
* Responsive Flask-based web application

# 🔄 Workflow

1. User uploads an image.
2. Flask sends the image to API Gateway.
3. API Gateway invokes AWS Lambda.
4. Lambda uploads the image to Amazon S3.
5. Amazon Rekognition detects labels in the image.
6. Lambda generates a descriptive sentence.
7. Amazon Polly converts the description into speech.
8. MP3 is stored in the audio S3 bucket.
9. Metadata is stored in DynamoDB.
10. Flask retrieves all records and displays them on the website.

# 📡 API Endpoints

## Upload Image

**POST**

```
/upload
```

### Request

```
{
    "filename":"dog.jpg",
    "image":"Base64EncodedImage"
}
```

---

## Get Images

**GET**

```
/images
```

### Response

```
{
  "images":[
      {
          "imageName":"dog.jpg",
          "description":"This image contains Dog, Grass and Tree.",
          "primaryLabel":"Dog",
          "highestConfidence":99.87,
          "audioURL":"...",
          "imageURL":"..."
      }
  ]
}
```

---

# 💾 DynamoDB Attributes

| Attribute         | Description              |
| ----------------- | ------------------------ |
| ImageId           | Unique Image ID          |
| ImageName         | Uploaded Image Name      |
| ImageURL          | S3 Image URL             |
| AudioURL          | Generated MP3 URL        |
| Description       | AI Generated Description |
| PrimaryLabel      | Main Detected Object     |
| HighestConfidence | Highest Confidence Score |
| Labels            | All Detected Labels      |
| Timestamp         | Upload Date & Time       |
